# Example Paper 001

This is the smallest development scaffold in the `packs/` directory. It exists to show the shape of an `agent_spec.json` file, not to translate a real research paper into a decision-ready governance artifact.

The pack is useful for schema inspection and early workflow testing. It should not be cited as a research translation, deployment assessment, or completed decision record.

## Current contents

`agent_spec.json` defines a draft specification with the core fields expected by the translator: decision scope, inputs, outputs, evaluation hooks, and governance controls. The pack does not include source text, extracted claims, task definitions, an evaluation plan, safety-policy intake, or a decision summary.

## Appropriate use

Use this pack when testing whether the repository can parse a minimal specification. Do not use it to support any claim about research quality, model behavior, operational readiness, or governance suitability.

## Maturity status

This pack is a Level 0 scaffold. It becomes a real translation pack only after source material, claims, tasks, evaluation criteria, and a human decision summary are added.
