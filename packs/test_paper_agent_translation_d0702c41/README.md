# Test Paper Agent Translation Pack

This is a development test pack for the translator workflow. It exists to check whether the repository can carry an agent specification and evaluation-plan stub through the expected file structure.

The pack should not be treated as a real research translation. It contains no preserved source text, no extracted claims, no task definitions, no safety-policy intake, and no decision summary.

## Current contents

`agent_spec.json` provides a draft system specification. `eval_plan.json` provides a minimal evaluation-plan shell. Together, they support schema and workflow testing.

## Appropriate use

Use this pack for development checks, parser tests, workflow experiments, and schema inspection.

Do not cite it as evidence. Do not use it to support any AI governance conclusion. Do not treat it as decision-complete.

## Maturity status

This pack is a Level 0 or Level 1 development artifact, depending on the test context. It requires source material, claims, tasks, evaluation criteria, and a human decision summary before it can become a governed research translation pack.
